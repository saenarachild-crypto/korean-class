김현석 국어 셀프해설 작성기 · 생성기

시험 정보와 정답을 입력하면, 학생이 오답 해설·근거·다음 판단 기준을 직접 작성하고 카톡 보고서로 제출하는 HTML을 만듭니다.

1. 기본 정보

시험마다 다르게. 학생 작성 내용이 시험별로 따로 저장됩니다.
영어·숫자·하이픈만. .html 자동.

2. 배포 · 링크 · 공유

학생 파일을 올리는 폴더 주소입니다. 폴더가 바뀌면 여기만 고치면 됩니다. og:url과 카톡 미리보기에 사용됩니다.
학생이 셀프해설을 제출한 뒤에만 보입니다.
비워두면 구글시트로 전송하지 않습니다.
카톡 썸네일과 공유 이미지. 공개 URL이어야 합니다.
학생이 카톡에서 링크를 받을 때 굵게 보이는 제목.
제목 아래 회색 글씨. 비워두면 부제가 들어갑니다.

3. 정답 입력

쉼표·공백·줄바꿈 모두 가능. 34개 필요.

4. 3점 문항 입력

비워두면 45문항 균등 환산 점수로 계산됩니다.

5. 생성

카톡 미리보기

입력하면 즉시 갱신됩니다. 처음 공유 시 미리보기가 안 뜨면 카카오 공유 디버거에서 캐시를 비우세요.

검사 결과

아직 생성 전입니다.

학생 화면 흐름

1. 이름·목표 입력
2. 선택과목 선택
3. 답 선택 + 확신도(◯△✕)
4. 문항별 셀프해설 작성 (틀린 문항 자동 펼침)
5. 점수 확인 + 카톡 보고서 제출
단일 HTML 파일입니다. 생성 기능은 인터넷 없이도 작동합니다.
