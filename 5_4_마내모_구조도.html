심리적 거리: 시간·공간·발생 확률·사회적 관계 등 차원에서 느끼는 주관적 거리감

편향의 문제점

해석 편향은 인지 과정에서 자연스럽게 나타나지만, 특정 수준에만 머무르면 텍스트를 부분적으로만 이해하게 됨.

⚠️ 상위 수준에만 머무를 경우
  • 주제·교훈 등 일반적 의미만 파악
  • 구체적 근거·논증 구조 간과
  • 피상적 이해 위험
⚠️ 하위 수준에만 머무를 경우
  • 구체적 사례·표현 방식 등 세부 사실에 집착
  • 전체 맥락 간과
  • 필자 의도 상실 위험
▼
해결: 균형적 읽기

편향적 읽기에서 벗어나기 위해 텍스트와의 심리적 거리를 조절해야 함.

구체적 방법

심리적 거리 먼 독자 → 관련 가상 상황 상상 유도 → 구체적 공감 형성

심리적 거리 가까운 독자 → 시야 넓히기 유도 → 객관적 관점 확보

🏮 [4~9] 도가의 감정 이해와 성인무정 논쟁
(가) 장자의 감정론

도가는 일반적으로 감정을 중요하게 다루지 않음. 유가는 도덕적 감정(측은지심·시비지심) 확충. 도가는 감정 배제 경향. 그러나 『장자』에는 감정 논의 존재 → 도가의 이상적 삶의 태도를 이해하는 단초.

정(情)의 정의

외부 세계와 관계 맺으며 자연스럽게 발생하는 심리 반응.
포함: 호오(好惡), 기쁨, 분노, 즐거움, 슬픔.

핵심: 호오(好惡)

인간은 호오에 따라 행동하는 경향. 호오가 지나치면 자연의 흐름을 제대로 보지 못함.

판단과 감정은 분리될 수 없다

시비(옳고 그름 판단)도 감정에 속함. 인지적 판단 + 감정적 느낌 = 긴밀 연결.

예1) 위험하다 판단 → 공포 느낌
예2) 죽음이 옳지 않다 판단 → 슬픔

판단 변화 → 감정 변화 가능
예: 부인의 죽음 → 처음엔 슬픔 → 자연 변화 과정이라는 판단 → 감정 변화

▼
과도한 감정의 문제

과도한 감정 → 근심·두려움 → 몸 손상 → 감정에 얽매이면 이상적 상태 아님.

성인무정(聖人無情)

장자: 성인의 특징 = 성인무정

혜시 비판: "성인도 인간인데 감정이 없을 수 있나?"

장자의 해명: 무정 = 감정 자체 없음 ❌, 감정에 얽매이지 않음 ✅ → 몸 해하지 않고 정신 소모 없음.

성인은 진지(眞知)(자연과 인간에 대한 참된 앎)를 지님. 자연 흐름 순응 → 불필요한 시비 판단 감소 → 과도한 감정 이어지지 않음.

▼
(나) 현학에서의 성인무정 논쟁

현학의 관심: 현실 정치보다 존재의 근원(유와 무의 관계, 존재의 유래) 탐구에 집중.

하안의 입장

자연의 운행 = 의도·목적 없음 → 성인도 자연을 닮아 외물·사태에 직면해도 감정 없음 (문자 그대로의 무정).

⚠️ 문제 발생: 공자는 제자의 죽음을 슬퍼했다는 경전 내용과 상충
왕필의 입장

"성인도 감정을 가진다" 다만 보통 사람과 다름. 성인은 신명(神明) 보유. 신명 = '무'를 꿰뚫어 보는 능력.

무(無) = 만물의 근거, 유(有)의 원칙, 만물의 이치. 성인은 이치를 통찰하므로 감정을 느껴도 감정에 휘둘리지 않음 → 본성·이치에 부합하는 감정 표현 가능.

공자의 슬픔 해석: 실제 슬픔 감정 존재 + 예의에 어긋나지 않게 표현됨
▼
곽상의 입장

사생독화설(自生獨化說): 만물은 '무'라는 단일 근원에서 유래하지 않음. 무 = 아무것도 없음(만물의 근거 아님).

각 존재는 각자의 성분의 이치(타고난 고유한 본분)에 따라 존재. 원래는 서로 깊이 관여할 이유 없음 → 감정 드러날 본래적 이유도 없음.

그러나 인간은 옳고 그름 판단(인지적 태도) → 호오 발생 → 몸 해칠 감정 생성.

해결: 성분의 이치 통달 + 순응 → 불안 감정 없는 상태(무정) → 부정적 감정 사라지고 평정심 지배.

공자의 슬픔 해석: 실제 내면 슬픔 없음. 다만 사회 구성원으로서 감정 표현 행위에 동참했을 뿐.
전체 논리 축 정리
구분하안왕필곽상
감정 존재감정 없음감정 있음 (신명으로 통제)내면 감정 없음 (평정심)
공자 슬픔 해석상충 (문제점)실제 슬픔 + 예에 맞음내면 무감정, 사회적 동참
🎯 핵심 정리
장자는 감정 자체를 부정하지 않고 '얽매이지 않음'을 강조. 현학자들은 각자의 철학 체계로 재해석:
하안(무감정) → 왕필(이치에 맞는 감정) → 곽상(평정심)으로 논쟁 전개.
💧 [10~13] 초소수성 금속 표면의 구현 원리
문제 상황: 금속 부식
  • 금속 산화 → 표면에 극성 수산기 분포
  • 극성 물질인 물이 금속 표면에 넓게 퍼짐
  • → 금속 부식 발생
해결 목표

물이 퍼지지 않게 만들기 → 초소수성 구현

▼
친수성

물과 친화력 큼 → 물 잘 퍼짐

소수성

물과 친화력 적음 → 물 잘 안 퍼짐

초소수성

소수성이 극단적으로 강화됨 → 젖음성 매우 낮음 → 물방울이 퍼지지 않고 흘러감

▼
젖음성

액체 방울이 고체 표면에 퍼지는 정도

계면

서로 다른 분자들이 접하는 경계면. 내부보다 불안정 → 에너지 높음.
계면 에너지 = 계면의 단위 면적당 에너지

핵심 원리

계면 에너지 ↑ → 계면 면적 ↓ → 젖음성 ↓

자연은 불안정성을 줄이는 방향으로 작용하므로, 계면 에너지가 클수록 계면 면적이 줄어든다.

계면 에너지는 고체·액체 종류에 따라 결정됨.

▼
초소수성 구현 방법 ①: 소수성 코팅
  • 소수성 작용기 포함 코팅제를 금속 표면에 얇게 코팅
  • 수산기 외부 노출 차단 → 젖음성 감소
한계

코팅만으로는 초소수성 구현 어려움 → 요철 구조 형성 추가 필요

▼
초소수성 구현 방법 ②: 요철 구조 형성
  • 금속 표면에 원기둥 돌기 균일 분포 → 돌기 사이 공기층 형성
  • 물방울이 돌기 윗면에만 접촉 → 캐시-백스터 상태
캐시-백스터 상태

물방울 ↔ 돌기 윗면만 접촉

  • 고체 면적 분율 ↓ → 실제 접촉 면적 ↓
  • 젖음성 ↓ → 초소수성 강화

고체 면적 분율 = 액체 접촉 면적 중 고체에 닿은 면적 비율
돌기 간격 ↑ → 감소 / 돌기 윗면 면적 ↑ → 증가

핵심 효과: 계면 에너지 자체는 그대로지만, 접촉 면적 감소로 계면 에너지가 커진 것과 같은 효과

▼
문제 발생: 웬젤 상태 전이
  • 돌기 사이 간격 너무 큼 또는 높은 압력 발생
  • 물방울이 공기층 밀어냄 → 돌기 사이 침투 → 웬젤 상태
웬젤 상태
  • 물방울이 돌기 사이까지 침투 → 계면 넓어짐
  • 젖음성 ↑ → 초소수성 사라짐

특히 높은 곳에서 떨어지는 빗방울(압력 큼)은 웬젤 상태 더 빠르게 발생

▼
추가 해결책: 소수성 나노 입자 활용

기존 해결책의 한계: 돌기 높이 증가 → 공기층 두꺼워짐 → 웬젤 상태 억제, but 돌기 마모 문제

대안: 소수성 작용기로 둘러싸인 소수성 나노 입자를 코팅제에 혼합

코팅 시 3가지 효과
  • ① 수산기 노출 차단 → 기본 소수성 강화
  • ② 돌기 표면에 미세 요철 형성 → 접촉 면적 추가 감소
  • ③ 공기층 붕괴 억제 → 웬젤 상태 전이 지연

최종 결과: 초소수성 강화 + 지속성 향상
