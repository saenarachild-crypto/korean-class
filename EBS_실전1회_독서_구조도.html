단락별 핵심어 중심 정리 — 김현석 국어
[1지문] 독서 공간의 의미와 확장
인문 · 1~3번
주제 독서 공간의 의미와 독서 공간 확장의 중요성
1문단 · 정의
독서 공간= 책을 읽고 있는 / 읽을 수 있는 공간 → 일반적으로 물리적 독서 공간(㉠)
본원적 독서 공간
본래 독서 위해 존재 · 서재(개인) / 도서관(대중)
전용된 독서 공간(㉡)
목적은 다름 · 책 읽는 곳으로 활용 · 버스 / 지하철
▼
2문단 · 구분의 모호함
개념상 명확히 구분 ↔ 실제론 구분 어려움
방(침실+서재), 카페·서점 → 전용된 공간 / 교실 ≈ 본원적(학습=독서 전제 시)
독서실= 이름은 본원적, 실제는 시험공부용 → 본원적 아님
▼
3문단 · 인식적 공간
인식적 독서 공간(㉢) = 의식 속에 형성 · 물리적 제약 ✕
수업 시간 · 집에 가는 길 · 아침 독서 운동 · 독서 모임
능동적 선택, 관계 속에서 새롭게 구성·확장
▼
4문단 · 가상 공간으로 확장
지식 정보화 사회 + 디지털 기술 발전 → 가상 공간속 독서 공간 무한 확장
비용·시간 제약 거의 없음 · 누구나 도달 · 읽기 + 쓰기 가능한 열린 공간
[2지문] 야스퍼스의 한계 상황과 실존
인문(철학) · 4~7번
주제 한계 상황을 통한 초월자와의 만남과 실존 회복
1문단 · 상황 내 존재
인간 = 상황 내 존재 · 상황에 대한 물음 = 철학의 근원
모든 상황은 인간이 스스로 선택한 것이 아님 · 삶을 짓누름
▼
2문단 · 한계 상황
한계 상황= 변경 불가 · 근원적 → 죽음·고통·투쟁·죄책감
소멸되지 않음 · 항상 함께 · 직면 시 회의·불안·좌절
▼
3문단 · 경험의 중요성(㉠)
한계 상황 경험 = 부정적 ✕ → 유한성 자각
외면 = 자기 포기 · 대응 방식이 중요
좌절 → 초월자를 감지하는 계기 (㉠의 이유 핵심)
▼
4문단 · 초월자와의 만남
세계 = 불완전 → 깨달을 때 초월자에게 다가감
언어로 규정 ✕ · 표상 인식 ✕ · 추론 도달 ✕
한계 상황 경험 = 실존 / 초월자 만남 없이 실존 ✕
▼
5문단 · 암호 해독 → 실존 회복
초월자는 가능적 실존에게 암호로 나타남
현실적 언어 ✕ · 다의적 · 하나로 고정 불가 · 각자의 방식으로 해독
암호 해독 과정 → 실존 회복
[3지문] 동시 다중 라운드 경매
사회(경제) · 8~11번
주제 동시 다중 라운드 경매 방식의 등장 배경과 성과
1문단 · 심사 방식(ⓑ)
경매 전 → 심사(사업 계획서 평가로 선정)
문제: 평가·이의 절차 → 할당까지 수년 소요(시간)
▼
2문단 · 제비뽑기(ⓒ)
1982~ 제비뽑기 → 시간·노력 대폭 단축
문제: 재판매 가능 → 투기꾼 참여 → 사용권 파편화(투기) → 전국망 사업자 곤란
▼
3문단 · 뉴질랜드 경매(1989)
기존 경매 방식 → 차순위 입찰가 지불(㉠) · 동시 1회 결정
타인 입찰 정보 모름 → 낙찰이 운에 좌우 · 가격 낮음 → 효율·수익 둘 다 실패
▼
4문단 · 동시 다중 라운드 경매(㉡)
폴 밀그럼 제안 (미국 1993~) · 모든 주파수 동시 경매
라운드마다 최고가·입찰자 공개 → 전략 수정 → 새 입찰 없을 때까지 반복
이전 라운드보다 많은 주파수 입찰 ✕ → 불공정 정보 행위 예방
▼
5문단 · 성과
합리적 낙찰 + 정부 수입 극대화
영국(2000) 성공 → 일반적 형태 정착 → 폴 밀그럼 등 2020 노벨 경제학상
[4지문] 초전도체와 초유체
과학(물리) · 12~17번
주제 오너스가 냉각 실험을 통해 발견한 초전도체와 초유체
1문단 · 발견 조건
물리계 ↔ 주변 환경 에너지 교환 (온도↑ = 교환↑)
거시적 물리계 → 온도 낮아야 양자 역학적 현상 발견
오너스 — 4.2K 헬륨 액화 성공 → 두 현상 발견(초전도·얼지 않는 액체)
▼
2문단 · 저항 견해 3인
절대 영도 근처 도체의 전기 저항?
마티센: 불순물·결함 있으면 저항 남음
듀어: 불순물 없는 도체 → 저항 완벽히 사라짐
켈빈: 전자 얼어붙음 → 저항 오히려 커짐
▼
3문단 · 초전도체(수은)
1911 수은 4.2K 냉각 → 전기 저항 0 = 초전도 현상
수은 선택: 낮은 온도에 이미 액체 → 불순물 효과 줄임
단일 원소 금속 29종 초전도 발현 (15종은 2K 미만, 도달 어려움)
▼
4문단 · 양자 액체(헬륨)
헬륨 고체화 시도 → 실패 (2.2K 아래서도 얼지 않음)
가벼운 비활성 기체 → 큰 영점 운동 · 구속할 힘 부족(㉠)
→ 절대 영도에서도 액체 = 양자 액체 · 2.2K 밀도·비열 최댓값(람다점 ㉡)
▼
5문단 · 초유체
1937 헬륨 II 정체 규명 → 2.2K 이하 상전이 · 점성 사라짐
저항 없이 흐름 → 초전도체와 닮음 → 초유체
